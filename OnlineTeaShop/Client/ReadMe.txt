1. Purpose of the project
The aim of the project is to show all the products that are available in our online tea shop.
For the convenience of customers, the products are divided into available and non-available.

2. Creating the project
Blazor WebAssembly framework is used to create the project. Bootstrap version 4.5.2 is used for the styles.
HTML5 and C# code are also used.
Three components have been created:
-index component -> the purpose is to show all products.
-OutOfStock component  -> the  purpose is to show only products that are not available.
-InStock component -> the purpose is to show products that are available.

3. Starting the project
In a command shell, navigate to the project folder. 
Run the following command: dotnet run.
Open the corresponding link in the browser http://localhost:5000.
